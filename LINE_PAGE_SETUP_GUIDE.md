# LINEページ 新寺院セットアップガイド

静蓮寺（joren-ji）形式のLINEページを他のお寺に展開するための手順書。
作業前に本ガイドを通読し、ヒアリングシートをすべて埋めてから実装に入ること。

---

## 1. ヒアリングシート（お寺から取得する情報）

実装開始前に以下をすべて確定させる。未確定のまま進めない。

### 1-2. LINE / LIFF URL

| 項目 | URL | 確認方法 |

| LINEメッセージURL | 例：`https://line.me/R/oaMessage/@103pffyg/?` | LINEの「公式アカウント管理画面 > プロフィール > シェアURL」 |

|---|---|---|---|

### 1-5. ご法事チェックリスト（タブ構成）

お寺によって内容が大きく異なるため、以下をヒアリングして確定させる。

**タブ構成（何を設けるか）：**
- 例：納骨・年忌法要・新盆
- 例：年忌法要・初盆・お彼岸

**各タブの「お持ち物」：**
- 必須項目（チェックボックス必須）
- 任意項目（「任意」バッジ付き）

**各タブの「事前連絡事項」：**
- 塔婆申込み の有無・料金（施主大塔婆・志主小塔婆）

- 名義変更のご印鑑 → 年忌法要で不要な場合が多い
- お位牌のサイズご連絡 → お寺で作成を扱わない場合は削除

## 3. 実装手順

### Step 1: ディレクトリとファイルのコピー

```bash
cp -R joren-ji/ {新しい寺院スラッグ}/
```

### Step 2: 画像ファイルの配置

取得した画像を `{slug}/images/` に配置し、ファイル名を命名規則に合わせる。

### Step 3: 全ファイルの一括置換（基本情報）

以下を全HTMLファイルで置換する（検索・置換で一括対応）：

| 置換前（静蓮寺の値） | 置換後（新しいお寺の値） | 対象ファイル |
|---|---|---|
| `浄土宗 紫金山 妙雲院 静蓮寺` | 新寺院の正式名称 | 全ファイル |
| `静蓮寺` | 新寺院の略称 | 全ファイル |
| `〒110-0004 東京都台東区下谷1-12-21` | 新住所 | 全ファイル |
| `03-3843-4034` | 新電話番号 | 全ファイル |
| `03-3843-3442` | 新FAX番号 | 全ファイル |
| `zen0801key@gmail.com` | 新メールアドレス | 全ファイル |
| `https://line.me/R/oaMessage/@103pffyg/?` | 新LINEメッセージURL | chat-introduction.html |
| `https://liff.line.me/2010126434-cKECWmxF?unique_key=RGS1oQ&ts=1779148721` | 新LIFF URL | houji/houji.html |
| `jorenji_hondo.jpg` | 新ヒーロー画像ファイル名 | houji/houji.html |
| `jorenji_carousel_1.jpg` | 新カルーセル1ファイル名 | event.html |
| `jorenji_carousel_2.jpg` | 新カルーセル2ファイル名 | event.html |
| `jorenji_carousel_3.jpg` | 新カルーセル3ファイル名 | event.html |
| `jorenji_priest.jpg` | 新住職写真ファイル名 | chat-introduction.html |

### Step 4: `<title>` タグの更新

各ファイルのタイトルを更新する。

| ファイル | 置換前 | 置換後 |
|---|---|---|
| houji/houji.html | `ご法事のご案内｜静蓮寺` | `ご法事のご案内｜{寺院名}` |
| houji/list.html | `ご法事前のチェックリスト｜静蓮寺` | `ご法事前のチェックリスト｜{寺院名}` |
| houji/flow.html | `当日の流れ｜静蓮寺` | `当日の流れ｜{寺院名}` |
| event.html | `年間行事のご案内｜静蓮寺` | `年間行事のご案内｜{寺院名}` |
| news.html | `新着情報｜静蓮寺` | `新着情報｜{寺院名}` |
| chat-introduction.html | `静蓮寺へLINEで相談する` | `{寺院名}へLINEで相談する` |

### Step 5: event.html の年間行事を更新

`<!-- ────── タイムライン ────── -->` 内の各 `.tl-row` を新寺院の行事に差し替える。

```html
<!-- 行事1件のテンプレート -->
<div class="tl-row">
  <div class="tl-left">
    <div class="tl-month">{月}<span class="unit">月</span></div>
    <div class="tl-line"></div>
  </div>
  <div class="tl-body">
    <div class="ev-card" data-date="{YYYY-MM-DD}">
      <div class="ev-name">{行事名}</div>
      <div class="ev-ruby">{ふりがな}</div>
      <div class="ev-date">{日程テキスト}</div>
      <div class="ev-desc">{説明文}</div>
    </div>
  </div>
</div>
```

> `data-date` には当年の代表日（例：`2026-05-24`）を入れる。JSが自動的に「次の行事」バッジを付与する。

### Step 6: news.html の記事を更新

`<div class="news-list">` 内の各 `.news-card` を新寺院の新着情報に差し替える。`data-category` の値は、タブの `data-filter` と完全一致させる（`all` 以外）。

```html
<!-- 記事1件のテンプレート -->
<div class="news-card" data-category="{お知らせ|行事・催し|その他}">
  <div class="news-img-placeholder">
    <!-- 既定の書類アイコンSVGをそのまま流用、または <img src="..."> に差し替え -->
  </div>
  <div class="news-content">
    <div class="news-meta">
      <span class="news-tag">{カテゴリ}</span>
      <span class="news-date">更新日：{YYYY.MM.DD}</span>
    </div>
    <div class="news-title">{記事タイトル}</div>
    <div class="news-excerpt">{抜粋テキスト}</div>
    <a href="{詳細URL or #}" class="news-btn">詳細を見る →</a>
  </div>
</div>
```

> 記事を追加・削除しても JS は自動で空表示メッセージ（「該当する記事がありません」）を切り替えるため、JS 本体は触らない。

> タブ構成自体を変えたい場合は `.tab-grid` 内の `data-filter` 値と各カードの `data-category` を必ず揃える。

### Step 7: houji/list.html のタブ・チェックリストを更新

お寺の方針に合わせてタブ構成・各項目を編集する。
詳細は `joren-ji/DESIGN_SPEC.md` を参照。

### Step 8: houji/flow.html のアクセス情報を更新

- GoogleマップiframeのURLを新住所に変更
- 「Googleマップで開く」の共有リンクを更新
- 住所・電話番号を更新

### Step 9: chat-introduction.html の受付情報を更新

住職名・返信目安・その他案内文をお寺の実態に合わせて修正する。

### Step 10: DESIGN_SPEC.md の作成

`joren-ji/DESIGN_SPEC.md` をコピーし、寺院固有の情報に書き換える。

### Step 11: AIレビューエージェントで反映漏れをチェック

`agents/line-page-reviewer/CLAUDE.md` の手順に従い、自動チェックを実行する。

## 5. よくあるミスと対策

| ミス | 対策 |
|---|---|
| `line://` を使ってしまう | 必ず `https://line.me/` または `https://lin.ee/` を使う |
| 画像パスが壊れている | `onerror` フォールバックがあるので目視では気づきにくい → AIレビューで検出 |
| `data-date` が去年の日付のまま | 毎年更新が必要。年始に確認する |
| 関係寺院（正定寺など）が残っている | 新寺院に関係寺院がない場合は連絡先ブロックごと削除 |
| 費用情報が更新されていない | 掲載しない場合はコストボックスを丸ごと削除する |
| 旧寺院名が `<title>` に残っている | ブラウザタブには出るが、普段目につきにくい → チェックリストで必ず確認 |
| news.html のタブ data-filter とカード data-category がズレ | 一方を変更したら必ずもう一方も合わせる。ズレるとそのタブで0件表示になる |
